# Vault: Expense Manager

A privacy-first, local-only wealth tracker built with Flutter.

### Architecture
- **Clean Architecture** (Data, Domain, Presentation layers)
- **State Management**: Riverpod
- **Local Database**: Isar

### Features
- Dual-stream accounting (Burn/Store/Income/Transfer)
- Smart category matching
- Analytics engine with charts
- Daily notification reminders
- Google Drive backup & restore
- Multi-currency support

### Tech Stack
- Flutter + Dart
- Riverpod for state management
- Isar for local database
- fl_chart for data visualization
- Google Sign-In & Google Drive API for backup
